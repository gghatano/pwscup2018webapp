= サンプルプログラム集 (2018-09-06)

== 概要
PWS Cup 2018 のルールに従った処理の例を一通り実装したサンプルプログラム集です．
正しい動作を保証するものではありませんが，コンテスト参加の手助けとなることを期待して公開します．

本サンプルプログラム集は，山口高康さんが作成された以下のプログラムを改変して，
加工から評価までを一通り実行できるようになっています．

----
drillプログラム ver.0.1
2016.7.13 Takayasu Yamaguchi

1) player.csv に参加者情報を記入する
2) command.sh を実行する
3) result.txt を閲覧する
----

動作確認は以下のソフトウェア上で行いました．
* cygwin 2.829
* ruby 2.3.6p384
* Python 3.6.4

== プログラムの入出力
山口さんのオリジナルと同じく標準入力から入力を読み込み，標準出力に出力しますが，入出力の形式が一部異なっています．
defense (加工): J(T) -> J(A)
publish (加工トランザクションから公開加工トランザクションへの変換): J(A) -> J(A')
offense (再識別): J(T, A') -> J(F')
security (安全性評価): J(R, T, A, F') -> J(S)
utility (有用性評価): J(T, A) -> J(U)

ここで，J(X, Y, Z) は，
 [X の行数]
 [X の中身]
 [Y の行数]
 [Y の中身]
 [Z の行数]
 [Z の中身]
を並べたデータを表します．

== 各プログラム (lib/*) の説明

=== システム関連
* pub.py: A から A' への変換
* sec-test.py: 安全性評価
* ut-il.py: 有用性評価

=== 加工
* ano-del-lc.py: T の各要素を確率指定された確率で削除
* ano-random.rb: T をランダムに一般化

=== 再識別
* re-1.py: A' で最初に出現する仮名を T の最初に出現する仮名と推測
* re-uniq.rb: 完全一致の行から推測
* re-item.py 商品からの推測

=== その他
* common.py: python 版サンプルの入出力部分
* common-*.rb: ruby 版サンプルの共通部分
